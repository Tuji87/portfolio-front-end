Sobre mim

Como desenvolvedor front-end e analista de dados, combino minha paixão por criar experiências digitais memoráveis com a habilidade de extrair insights valiosos através da análise de dados. Atualmente, faço parte do programa ONE - Oracle Next Education, uma iniciativa que tem fortalecido minha base técnica e ampliado minha visão sobre desenvolvimento web, com foco em HTML, CSS e React.

Trabalho em parceria com pequenos negócios e designers, utilizando tanto minhas habilidades em desenvolvimento front-end quanto em análise de dados para criar soluções completas e baseadas em evidências. Cada projeto é uma oportunidade de aplicar conhecimentos técnicos e análises aprofundadas, combinando criatividade com dados para entregar resultados excepcionais.
